## How to publish on GitHub Pages

1. Create a new GitHub repository.
2. Upload all files and folders from this package.
3. Go to Settings.
4. Open Pages.
5. Under Build and deployment, choose Deploy from a branch.
6. Select the main branch and the root folder.
7. Save.
8. Wait for GitHub to generate the website link.

## How to update images

Replace images inside assets/img with the same file names.

## How to update project text

Edit index.html for the homepage.
Edit each file inside the projects folder for project pages.
